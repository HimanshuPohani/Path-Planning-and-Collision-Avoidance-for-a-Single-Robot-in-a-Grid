# Path-Planning-and-Collision-Avoidance-for-a-Single-Robot-in-a-Grid
Project Title: Path Planning and Collision Avoidance for a Single Robot in a Grid

How to Run:
1. Make sure Python is installed.
2. Install dependencies:
   pip install numpy matplotlib
3. Run the file:
   500120393 DCN PROJECT.ipynb

What it does:
- Generates a 10x10 grid with random obstacles.
- Finds a collision-free path using A* algorithm.
- Visualizes the grid, path, and obstacles.
